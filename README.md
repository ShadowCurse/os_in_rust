# OS in Rust

This is an implementation of the [Phil-Opp](https://os.phil-opp.com/minimal-rust-kernel/) blog post series.

## Building and runnig

To test the project
```sh
$ cargo ktest
```

To build the project
```sh
$ cargo kbuild
```

To build the image
```sh
$ cargo kimage
```

To build and run the image
```sh
$ cargo krun
```
